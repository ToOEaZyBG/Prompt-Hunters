# Frontend Structure Documentation

## Основни страници (pages):
1. Home - Начална страница
   - Showcase на най-популярните промптове
   - Категории промптове
   - Търсачка

2. Browse - Разглеждане на промптове
   - Филтри по категории
   - Сортиране (по цена, популярност, дата)
   - Списък с промптове

3. PromptDetails - Детайли за промпт
   - Информация за промпта
   - Примери за използване
   - Рейтинг и ревюта
   - Бутон за покупка

4. Auth - Автентикация
   - Логин форма
   - Регистрация
   - Забравена парола

5. UserDashboard - Потребителски панел
   - Закупени промптове
   - Създадени промптове
   - Профил настройки
   - Баланс и транзакции

## Основни компоненти (components):
1. Layout
   - Navbar
   - Footer
   - Sidebar (за филтри)

2. PromptCard
   - Картичка за показване на промпт в списъци

3. SearchBar
   - Търсачка с филтри

4. Filters
   - Категории
   - Ценови диапазон
   - Рейтинг

5. UserMenu
   - Профил меню
   - Нотификации
